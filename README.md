# Disease Prediction from Symptom Profiles

An end-to-end machine learning project that predicts disease diagnoses from binary symptom
vectors. Five models are compared (Logistic Regression, Decision Tree, Random Forest,
XGBoost, MLP Neural Network), with SHAP explainability and an interactive Streamlit app.

## Folder Structure

```
DiseasePredict/
├── Training.csv              # 4,920 rows × 133 columns (132 symptoms + prognosis)
├── Testing.csv               # 42 rows × 133 columns (held-out test set)
├── disease_prediction.py     # Full pipeline: EDA → models → SHAP → saves artifacts
├── app.py                    # Streamlit app (loads saved models)
├── requirements.txt
├── model_comparison.csv      # Generated by disease_prediction.py
├── figures/                  # All saved .png visualizations
└── models/                   # All saved model files
    ├── label_encoder.pkl
    ├── logistic_regression.pkl
    ├── decision_tree.pkl
    ├── random_forest.pkl
    ├── xgboost.pkl
    ├── mlp_model.keras
    ├── mlp_scaler.pkl
    ├── top20_shap_features.pkl
    └── best_hyperparams.pkl
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Training Pipeline

This script performs EDA, trains all five models with cross-validation, generates all
figures, computes SHAP values, and saves every artifact to `figures/` and `models/`.

```bash
cd /path/to/DiseasePredict
python disease_prediction.py
```

Expected runtime: ~5–20 minutes depending on hardware (XGBoost GridSearch is the bottleneck).

## Run the Streamlit App

The `models/` folder must exist and be fully populated before running the app.

```bash
streamlit run app.py
```

Then open [http://localhost:8501](http://localhost:8501) in your browser.

## Dataset Notes

- **Target**: `prognosis` — 41 disease classes, perfectly balanced (120 samples each)
- **Features**: 132 binary symptom columns (0 = absent, 1 = present)
- **Training**: 4,920 rows | **Test**: 42 rows (one per class, true held-out set)
- **Missing values**: None (trailing `Unnamed: 133` column dropped on load)
- **random_state=42** used everywhere for reproducibility
